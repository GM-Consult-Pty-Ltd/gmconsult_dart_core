LanguageTagStringExtensions

LanguageTagStringExtensions extension Null safety

Extension methods on String to parse a Unicode Language Identifier to a its subtags.

on
  • String

Properties

countrySubTag → String?
Parses the String to subTags and returns the country subTag if it exists.
read-only
languageSubTag → String
Parses the String to subTags and then returns the language subTag if it exists.
read-only
scriptSubTag → String?
Parses the String to subTags and then returns the script subTag if it exists.
read-only
subTags → List<String>
Splits the string into sub tags at hyphens, em-dashes and/or underscores.
read-only

Methods

normalizeLanguageIdentifier() → String
Replaces all sequences of dashes, em-dashes and underscores with a single dash, then strips out all characters other than letters and dashes.
