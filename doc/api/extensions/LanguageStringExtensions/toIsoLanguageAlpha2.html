String toIsoLanguageAlpha2()

Returns an ISO 639-1 2-character language code if the String is a valid language code.

Converts string to lower case, replaces hyphens with underscores, returns the language code after checking it consits of two lower case letters except when language is "und".

Returns 'und' if the String cannot be parsed to a valid 2-charcter ISO language subtag.

Implementation

String toIsoLanguageAlpha2() {
  if (isValidLanguageString) {
    final language = subTags.first.toLowerCase();
    return (RegExp(r'^[a-z]{2}$').hasMatch(language)) ? language : 'und';
  }
  return 'und';
}
